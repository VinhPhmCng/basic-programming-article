Syntax 10.3. Calling a function.

Depending on how a function is declared, we have two ways to call it:
\begin{equation*} {\color{blue} \text{function's name}} \, \text{(input 1, input 2, ...)} \end{equation*}
or
\begin{equation*} \text{input 1} . \! {\color{blue} \text{function's name}} \, \text{(input 2, input 3, ...)} \end{equation*}
